I am assuming the git is already installed on the server. But in case it isn't intalled on your local machine, that's the first thing we need to do:

$ sudo apt-get install git

Create a wee repository on a server other than Github. First we need to make a directory on your local machine. Next we need to initialise it as a git repository.

$ mkdir project_name
$ cd project_name
$ git init

Then we need to clone it into a bare repository.

$ cd ../
$ git clone --bare project_name project_name.git

Ignore the warnings: Cloning into bare repository 'project_name.git'... warning: You appear to have cloned an empty repository. done.

$ scp -r project_name.git user@xxxx.xx.xx.xx:/opt/git/
$ rm -r project_name project_name.git
$ git clone project_name.git

warning: You appear to have cloned an empty repository.

$ touch test.txt
$ git add test.txt
$ git commit -m "Testing git repo."
$ git push origin master
