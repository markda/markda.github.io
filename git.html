Create a wee repository on a server other than Github. First we need to make a directory on your local machine.

$ mkdir project

Next we need to initialise it as a git repository.

$ cd project
$ git init

Then we need to clone it into a bare respository.

$ cd ../
$ git clone --bare project project.git

Ignore the warnings: Cloning into bare repository 'project.git'... warning: You appear to have cloned an empty repository. done.

$ scp -r PROJECT.git user@xxxx.xx.xx.xx:/opt/git/
$ rm -r PROJECT PROJECT.git
$ git clone PROJECT.git

warning: You appear to have cloned an empty repository.

$ touch test.txt
$ git add test.txt
$ git commit -m "Testing git repo."
$ git push origin master
